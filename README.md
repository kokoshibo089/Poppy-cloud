# GitLinux Cloud IDE

GitLinux is a Cyberpunk-themed Cloud IDE and Terminal platform.

## Features
- Full Linux Terminal (via Xterm.js)
- Monaco Editor (VSCode Engine)
- Real-time WebSocket streaming
- Docker-based sandboxing

## Deployment
This repository is pre-configured for:
- Koyeb
- Hugging Face Spaces
- Render
- Oracle Cloud

Developed by GitLinux Engineering Team.
